VisionOS — Linux portable tarball
=================================

  tar -xzf VisionOS-*-linux-x64.tar.gz
  cd VisionOS-*-linux-x64
  ./install.sh

Installs to ~/.local by default. Options: --system, --prefix DIR, --uninstall

Runtime libraries and distro-specific commands:
  see the main README.md → "Linux — portable tarball"
